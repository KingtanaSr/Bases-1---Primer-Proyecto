Este sistema es una web diseñada utilizando React, node.js y Express, esta web permite gestionar y consultar ciertos datos de una base de datos de publicaciones sobre distintos temas relacionados con la biodiversidad. 
Las principales funcionalidades de este sistema son las de realizar consultas de publicaciones en base a el nombre científico de las especies sobre las que se tratan las publicaciones, también el poder crear, consultar, editar y eliminar publicaciones, así como mostrar las publicaciones agrupadas por colección.
Como ya se mencionó anteriormente, las tecnologías utilizadas para el desarrollo de esta aplicación fueron Node.js para manejar las rutas y controladores, PostgreSQL 16 para almacenar la información de las publicaciones y otros, y React más que todo para la apariencia (frontend) de la web.
